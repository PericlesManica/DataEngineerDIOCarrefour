# Projeto Bootcamp DIO / Carrefour Data Engineer



## Desafio GCP Dataproc



​	Neste projeto foi criado um ecossistema de Big Data usando o Google Cloud Platform (GCP), configurando o Google Cloud Dataproc e um Hadoop totalmente gerenciado.

​	Após a criação de todo o ecossistema, foi feita a inserção de um arquivo (livro.txt), onde foram extraidos 2 resultados/arquivos através de uma Job, o arquivo part-00000 contém uma lista de palavras e quantas vezes ela é repetida e o arquivo resultado.txt contém as 10 palavras que mais são usadas no livro.txt.

Obs.: todo o projeto foi concebido utilizando o navegador Google Chrome.
